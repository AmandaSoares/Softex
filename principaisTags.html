Metadados

São informações sobre a página que não são visíveis, mas ajudam a descrever seu conteúdo e otimizar a pesquisa.Por exemplo, vincular folha de estilo(css) com a tag link.

<link rel="stylesheet" href="styles.css">

Separação de conteúdo

É uma prática importante para organizar e estruturar o seu código de maneira clara e compreensível. Isso facilita a manutenção, a colaboração e a compreensão do código por outras pessoas.Exemplo de tags de separação de conteudo:

<header> definir o cabeçalho de uma seção ou de todo o documento

<nav> agrupar links de navegação, como menus, barras de navegação e links para outras páginas ou seções do site

<section> definir uma seção autônoma e significativa de conteúdo

<article> marcar um conteúdo independente e autocontido em uma página da web

<aside> marcar um conteúdo secundário ou complementar em uma página da web

<footer> definir o rodapé de uma página da web

Imagem

É usada para exibir imagens na página da web. A tag <img> não tem fechamento, pois é uma tag vazia.

É uma estrutura que permite aos usuários inserir dados interativamente em uma página da web e enviá-los para um servidor para processamento.
